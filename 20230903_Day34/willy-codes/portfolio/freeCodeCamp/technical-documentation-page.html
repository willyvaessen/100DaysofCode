Color scheme

During the Novi bootcamp (Full Stack Developer) we were pointed in the direction of Coolors for generating color schemes. At the time it was possible to enter a color and generate a palette based on that color. To do that, I wanted to use my own date of birth (1 October 1978) to generate that palette.

Using that date as a hexadecimal code #011078 ends up in the color Phthalo blue, which I liked a whole lot better that what came out of the HSL value of the date, #cdc1c1 being Silver. A nice color in itself, but not my taste, so I went for the Phthalo blue.

The palette that originated from that first color contains the following colors:

  • Phthalo blue (#011078) — mostly used as background color for my site footer and the main font color.
  • Glaucous (#6D769A) — used as the color for my social media link icons.
  • Cadet gray (#6D769A) — not used in the website design at this time.
  • Ash gray (#BEC2B4) — used in the gradient that fills the body background.
  • Bone (#CCCFB8) — not used in the website design at this time.
  • Pearl (#D3D5BA) — Not used in the website design at this time. Replaced by Alabaster.
  • Alabaster (#E6E7D8) — used for text on the darker background.

For the body background, I use a gradient: background: linear-gradient(315deg, #caceb1, #bec2b4, #b1b8b3, #9fa49e, #98968b) fixed; which consists of the colors Sage, Ash Gray (three variations) and Battleship gray

Typography

You might be able to discover two different fonts used on my website, current as well as previous versions. These fonts are:

  • Cantora
  • GlassTTY: TrueType VT220 font

The Cantora ('Singer' in Spanish) font, designed by Pablo Impallari is a friendly semi formal, semi condensed, semi sans serif. It has reminiscences of hand lettering, mixing straight and bowed stems, and natural curves.

According to the creator's website, it was born as an experiment in drawing from the outside to the inside (drawing the space surrounding the letters first, instead of drawing the letters themselves) in trying to apply the ideas and methods of Michael Harvey and Evert Bloemsma to my own glyphs construction.

I came across this font somewhere in December 2020 while looking for a font that allowed me to merge the V and W on top of each other. Cantora was the font that made this possible.

The GlassTTY: TrueType VT220 font is a font that mimics an ld school/retor CRT monochrome display. It is created by Viacheslav Slavinsky. My initial idea with this font is to create a website that feels and works like a BBS system that was used in the early days online communication.

While looking up the information on the fonts used, I came across the font 'Raleway' which I liked instantly, so I decided to use that font for this specific page.
